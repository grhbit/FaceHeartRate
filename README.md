# Face HeartRate

## Getting Started

### Prerequisites

- Xcode 10+
- Cocoapods
- node

### Installing and running

Build iOS Application
```sh
$ yarn
$ cd ios
$ pod install
$ open faceheartrate.xcodeproj
# Open Xcode project and build and run
```

Run Expo server
```sh
$ yarn # install dependencies
$ yarn start
```
